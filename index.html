What Is It

Turning willingness into a searchable network.

Blood cannot be manufactured — it can only come from another person. Yet most willing donors and the people who need them never find each other in time. This platform closes that gap: a place to register as a donor and to search for one, organised by exactly where in Nepal you are.

Who can donate

General eligibility guidelines followed by most blood banks in Nepal.

Age 18–65 years old
Weight 50 kg or more
Haemoglobin 12.5 g/dL or higher
At least 90 days since the last donation
Generally in good health, free from infection

How It Works

Five steps, register to reach.

Every donor and every search on this platform passes through the same short pipeline.

01

Register

A donor fills in their name, phone number, blood group, and exact location — province, district, and local level, down to the ward.

02

Validate

The form checks the phone number and blood group before anything is saved, so records stay clean and contactable.

03

Store locally

Details are saved in the browser's own storage using the Web Storage API — no server, no account, no external database.

04

Filter and match

Anyone in need searches the same list by province and blood type, and sees only donors who actually match.

05

Connect

A tap on a donor's number opens the phone dialer directly — the platform's job ends the moment two people can talk.

Key Highlights

Built to be simple, local, and fast.

Instant

Real-time registration

A donor is added to the list the moment the form is submitted — no waiting, no approval queue.

Nationwide

Full administrative hierarchy

Every donor is placed accurately by Nepal's 7 provinces, 77 districts, and local level type.

Targeted

Province and blood-type search

Filter the entire donor list down to exactly who's needed, in exactly the right area.

Private

Nothing leaves the browser

No account, no server, no third party — donor details stay stored locally on the device that entered them.

Direct

One-tap calling

Every donor's number is a live link that opens the phone dialer immediately.

Responsive

Works on any screen

From a classroom projector to a phone in someone's hand during an emergency.

Under The Hood

Technologies used.

No frameworks, no build step, no dependencies to install — just the core web platform, used deliberately.

Markup

HTML

Semantic, single-file structure that holds every section of this presentation and the donor tool itself.

Styling

CSS

Custom properties, Grid and Flexbox, and keyframe animation drive the layout, the theme system, and the pulse-line motion.

Logic

JavaScript

Vanilla JavaScript handles form validation, cascading province-district selects, filtering, and rendering — no library required.

Storage

Web Storage API

Donor records persist between visits using localStorage, entirely on the user's own device.

Typography

Google Fonts

Space Grotesk, Inter, and JetBrains Mono are loaded from the Google Fonts CDN for display, body, and data text.

Layout

Responsive Design

Fluid grids, clamp-based type, and three breakpoints keep the experience solid from a phone to a classroom projector.

What's Next

Where this project can grow.

A first version is deliberately small. These are the directions planned for future iterations.

Sync

Cloud backend

Move storage from the browser to a shared server so donor records aren't limited to one device.

Trust

Phone verification

An SMS OTP step when registering, to confirm numbers are real and reduce false entries.

Urgency

Emergency broadcast

Push an urgent request straight to matching donors nearby instead of waiting for someone to search.

Integration

Blood bank inventory

Connect with local blood banks and hospitals to show live stock levels alongside individual donors.

Reach

Nepali language support

A full Nepali-language interface so the platform is usable by a much wider audience.

Care

Donation history

Track each donor's last donation date and gently remind them once they're eligible again.

Thank You

Questions are welcome.
